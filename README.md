Food Delivery Analysis Dashboard

Overview
Analyzed food delivery data using Python and built an interactive Power BI dashboard to identify factors affecting delivery time.

Tools
* Python (Pandas, Matplotlib, Seaborn)
* Power BI

Key Insights
* Traffic significantly increases delivery time
* Distance is the strongest factor
* Bad weather causes delays
* Preparation time impacts performance

Dashboard

Conclusion
Traffic, distance, and preparation time are the main causes of delays. Optimizing these can improve delivery efficiency.
